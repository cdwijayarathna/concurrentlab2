To Run Serial program, first compile it using "gcc -o serial ll_serial.c" on your linux terminal.
Then run it using "./serial" . You will be asked to enter necessaru parameters at the run time. 

To Run Parallel program with one mutex, first compile it using "gcc -pthread -o with_mutex ll_with1mutex.c" on your linux terminal.
Then run it using "./with_mutex" . You will be asked to enter necessaru parameters at the run time.

To Run Parallel program with with read-write locks, first compile it using "gcc -pthread -o with_rwl ll_rwl.c" on your linux terminal.
Then run it using "./rwl" . You will be asked to enter necessaru parameters at the run time.
